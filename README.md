# Personal Website

## Tech Stack

- [Reactjs]
- [TailwindCSS]
- Library [react-awesome-reveal] for animations
- [fontawesome] for icons
- Using font from [google fonts]
  .

## Installation

Install the dependencies and devDependencies and start the server.

```sh
cd PersonalWebsite
yarn install
yarn start
```
